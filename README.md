# Sistek Monitoring System

A modern Spring Boot dashboard for monitoring production processes in real time. The system provides a clean and user-friendly interface to monitor PLC status, production lines, and barcode information.

## 🚀 Features

- 📊 Modern dashboard interface
- 🔌 PLC monitoring
- 🏭 Production line monitoring
- 📦 Barcode tracking
- 📈 Real-time status overview
- 🎨 Responsive and modern UI
- ⚡ Built with Spring Boot & Thymeleaf

## 🛠️ Technologies

- Java
- Spring Boot
- Spring MVC
- Thymeleaf
- HTML5
- CSS3
- JavaScript
- Maven

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/tuanaozkaaan/sistek-monitoring-system.git
```

Navigate to the project:

```bash
cd sistek-monitoring-system
```

Run the application:

```bash
./mvnw spring-boot:run
```

The application will be available at:

```
http://localhost:8080
```

## 📌 Project Purpose

This project was developed to provide a centralized monitoring dashboard for industrial production environments. It enables operators to monitor production data, PLC information, and barcode operations from a single interface.
